# myhealth-service
Backend microservice for myhealth POC

## Commands

### Generate swagger documentaton
```sh
make swag-v1
```

### Run Docker compose
```sh
make compose-up
```

### Tear down
```sh
make compose-down
```

### Run unit tests
```sh
make test
```

### Run integration tests
```sh
make integration-test
```
